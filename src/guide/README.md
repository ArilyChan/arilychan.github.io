# 指令

以指令的方式调用小阿日的各项功能

*在本文档右上角的搜索框输入具体指令可以快速跳转*
## 前缀
若未标注，目前可用`!`, `！`, `~`或不加任何前缀触发指令。

## 帮助
由于近期更新频繁，本网站可能有一些未标注的功能。请使用`help`或`帮助`命令查看。
一些功能也会在某些群组被禁用。若发现命令无法被触发，可能是由于此原因。更多细节请咨询群管理员或开发团队。
