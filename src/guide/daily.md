# 运势系统

> 今天也是大凶

倾向于osu玩家的运势系统，运势活动基本与osu相关

- 指令前缀： `!`

- `今日运势` 不需要指令前缀

- 以下指令说明中，以`<>`标注的为必要参数，`[]`为可选参数

**仅供娱乐，切勿当真**

## 今日运势

查看你的当天运势

- 指令： `今日运势`

- 无参数

*小阿日所在时区可能和您所在的时区不同，运势结果为小阿日所在的时区*

## 添加活动

向活动库里添加活动

- 指令： **!添加活动** `<活动名称>` `<宜详情>` `<忌详情>`

- 示例： `!添加活动 刷pp 刷了好多pp 刷了好多pc`

添加成功后，该示例运势活动表现为：
```
宜：刷pp
  刷了好多pp
```
或
```
忌：刷pp
  刷了好多pc
```

## 删除活动

⚠ 需要权限

从活动库里删除活动

- 指令： **!删除活动** `活动名称`

- 示例： `!删除活动 刷pp`

## 查看活动

查看活动库里指定的活动

- 指令： **!查看活动** `<活动名称>`

- 示例： `!查看活动 刷pp`

## 待审核 / 查看审核

查看审核中活动

- 指令： **!待审核** / **!查看审核**

- 无参数
